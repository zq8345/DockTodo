# DockTodo Pro

DockTodo Pro 是一个本地运行的高级任务管理应用，按滴答清单高级版的核心能力做成了离线可用版本。

## 使用

直接打开 `index.html` 即可使用。数据保存在浏览器本地，不需要服务器或安装依赖。

## 高级功能

- 今天、最近 7 天、日历、收集箱、全部、已完成视图
- 自定义清单（可重命名、改色、删除）和自定义筛选器
- 完整日历月视图
- 任务开始日期、截止日期、提醒（到点弹系统通知，未授权时页内提示）
- 重复任务：每天、工作日、每周、每月，完成后自动顺延并保留完成记录
- 标签：`#标签` 快速添加语法、侧栏标签视图
- 全局搜索：标题、备注、检查项、标签、清单名
- 优先级、备注、检查项和检查项提醒
- 预计番茄、实际番茄和专注计时
- 完成率、逾期、番茄进度统计
- 任务历史记录
- 多主题切换
- 本地数据导出（剪贴板/JSON 文件）与导入（合并或替换）
